# video-game-sales-tableau
This project presents an interactive Tableau dashboard analyzing global video game sales across genres, platforms, publishers, 
and time (1980–2020). The goal is to uncover sales trends, top-performing games, platforms, and publishers, while demonstrating 
strong data visualization and business intelligence skills using Tableau.

The dashboard allows users to explore:
Sales performance by genre
Sales evolution over time
Top-selling games, platforms, and publishers
Dataset-wide KPIs and summaries

Key Business Questions Answered
Which genres generate the highest global sales?
How have video game sales changed over time?
What are the top 10 best-selling games of all time?
Which platforms dominate global sales?
Who are the top publishers in the gaming industry?
How do sales differ when filtering by genre and date range?
